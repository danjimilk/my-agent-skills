# 작성 스킬 차단 (Glossary 미비)

만약 현재 에이전트가 실행하려는 스킬의 Frontmatter에 `type: write`가 설정되어 있다면, 문서 작성을 시작하기 전 프로젝트 내에 **용어집(Glossary, 예: `.docs/glossary.md` 등)**이 존재하는지 반드시 확인해야 합니다.

**용어집이 존재하지 않는다면, 에이전트는 해당 스킬(write)의 작업을 즉시 중단(Refuse)해야 합니다.**
중단 후 사용자에게 "용어집(Glossary)이 존재하지 않아 문서를 작성할 수 없습니다. 먼저 용어집을 생성해 주세요."라고 안내하세요.
